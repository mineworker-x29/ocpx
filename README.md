# ocpx

`ocpx` is an academic library for object-centric process intelligence.

## Scope
- OCEL 2.0 as the standard input data model
- analytical computation for object-centric process mining
- performance / conformance / context-oriented outputs
- PM4Py as IO backend
- OCPA-inspired algorithms (modernized)
- Research-grade reproducibility
- Benchmarks & CI included

## Status
Early research prototype.

## Quickstart

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .[dev]
pytest
ruff check .
```
